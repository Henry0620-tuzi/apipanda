01 / 项目介绍

先把事情
讲清楚。

ApiPanda 不是一个“匿名、无限、无门槛”的免费接口。我们计划建立的是一个公开透明、可持续运营的 AI API 接入层。

底层使用开源的 Sub2API 项目，统一不同模型供应商的调用方式。服务正式上线前，会先完成供应商授权、成本核算、风控和数据保护设计。

03 / FAQ

你可能
想知道。

现在可以注册或调用 API 吗？＋

还不可以。当前页面只是项目介绍和建设进度页，没有开放 API，也不会要求你提交上游密钥。

为什么要先准备域名、VPS 和上游？＋

域名用于稳定访问，VPS 用于运行服务，合规上游则决定服务是否能够合法、稳定地提供。三者准备好后再进入部署阶段。

项目会支持哪些模型？＋

暂未确定。最终以获得合法授权、成本可控且稳定可用的供应商为准，不会在授权不明确的情况下提供模型。

如何关注项目进展？＋

可以通过页面上的 GitHub 链接关注仓库。发布前会补充具体仓库地址、更新日志和服务状态页面。
