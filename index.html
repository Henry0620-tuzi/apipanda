FAQ

你可能想知道的事

关于 ApiPanda 当前阶段和未来计划的简要说明。

现在可以注册或调用 API 吗？＋

还不可以。当前只是项目介绍和建设进度页，没有开放 API，也不会要求你提交任何上游密钥。

为什么要先准备域名、VPS 和上游？＋

它们分别关系到稳定访问、服务运行和合法授权。准备完整后再部署，才能更可靠地提供服务。

以后会支持哪些模型？＋

最终以获得合法授权、成本可控且稳定可用的供应商为准，不会在授权不明确时提供模型。

如何关注项目进展？＋

可以关注我们的 GitHub 仓库。后续更新日志、部署文档和服务状态都会逐步公开。
